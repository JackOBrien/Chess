Chess
=====

Chess project for CIS 350-01  

**NOTES FOR THE TEAM:**

**Features:**  
* Notifies the players who is in check.
* Highlights valid moves 
* Blur and darken board when not in focus
* Pieces react to hover
* Pawn Promotion
* Pawn En Passant
* Casteling
